GPU Location Board

How does this app work?

  1. Select your motherboard model from the dropdown menu above.
  2. Click "Detect Current GPUs" to automatically detect all NVIDIA GPUs currently installed in your system.
  3. The app will show each PCIe slot on your motherboard and display which GPU (by index and model name) is detected in each slot.
  4. If you have an issue with a specific GPU (e.g., GPU 3), use this board to find the exact physical slot where that GPU is located in your system.
Tip: Use this tool to quickly identify and troubleshoot GPU placement, swaps, or hardware issues. The slot numbers correspond to the physical PCIe slots on your motherboard.
